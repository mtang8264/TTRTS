# TTRTS Design Document

## Concept

### Gameplay

* Cooperative gameplay
    * Competative version?

#### Playcycle
- Build a unique board by placing tiles from a deck
    - Tiles expand in complexity as you venture further from your starting point
    - Tiles can contain resources, points of interest, or hazards
- Construct structures on board tiles
    - Defensive features
    - Resource harvesters
    - Unit generators
- Fight against enemies

#### Game Pieces

* Character cards
    * Unique abilities
    * Specific cards that they get access to?
* Terrain tiles
    * Seperated into different difficulties so the game gets harder the longer you play
* Structure cards
    * Placed on terrain tiles to show what is built at a location
    * Have some effect (active or passive)
* Unit markers
    * Friends and foes
    * Able to move around terrain
    * Serve different functions like harvesting, building, or fighting

### Aesthetic

* Over the top grimdark
* Aspects of retrofuturism
